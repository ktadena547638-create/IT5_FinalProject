# D-INVENTY PRO™ - Icon Instructions
# ==================================

# Place your custom application icon here named 'icon.ico'
#
# REQUIREMENTS:
# - Format: ICO (Windows Icon)
# - Recommended sizes: 16x16, 32x32, 48x48, 256x256 (all embedded in one .ico)
# - Color depth: 32-bit (with alpha transparency)
#
# FREE TOOLS TO CREATE .ICO FILES:
# --------------------------------
# Online:
#   - https://icoconvert.com/
#   - https://convertico.com/
#   - https://www.favicon-generator.org/
#
# Desktop:
#   - GIMP (free): File > Export As > Select .ico format
#   - IcoFX (free trial): https://icofx.ro/
#   - Photoshop: Use ICO plugin from https://www.telegraphics.com.au/sw/
#
# DESIGN TIPS:
# ------------
# 1. Use a simple, recognizable design (inventory/box icon works well)
# 2. Ensure good contrast - icon should be visible on any background
# 3. Test at small sizes (16x16) - details get lost
# 4. Use your brand colors (#3b82f6 blue accent)
#
# AFTER CREATING:
# ---------------
# 1. Save the icon as: assets/icon.ico
# 2. Run: .\build.ps1
# 3. The icon will be embedded in the executable
#
# TEMPORARY SOLUTION:
# -------------------
# If you don't have an icon yet, the build will still work.
# The executable will use the default Python icon.
